# Job Portal System

A full-stack Job Portal System built using Java, Servlets, JSP, and MySQL.

## Features
- User registration and login
- Recruiter job posting
- Job seeker job search and apply
- Role-based access (Recruiter / Job Seeker)
- MySQL database integration

## Tech Stack
- Java
- Servlets & JSP
- MySQL
- HTML, CSS
- Apache Tomcat
- Eclipse IDE

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Set up MySQL database
4. Run on server
